🎨 Marcellino F.A – Designer Portfolio

Modern Designer Portfolio built with Next.js, TypeScript, Tailwind CSS, Shadcn UI, and NeoBrutalism UI.

---

✨ Overview

Marcellino F.A Portfolio is a modern and visually striking personal portfolio website designed for Graphic Designers, Brand Designers, Creative Directors, UI Designers, and other creative professionals.

Built using the latest web technologies and inspired by the Neo Brutalism design trend, this portfolio focuses on strong visual hierarchy, bold typography, colorful layouts, and excellent user experience.

📦 Installation

Clone repository:

git clone https://github.com/yourusername/marcellino-portfolio.git

Enter project directory:

cd marcellino-portfolio

Install dependencies:

npm install

Start development server:

npm run dev

Open:

http://localhost:3000

---

🏗 Build Production

npm run build

Run production:

npm start
