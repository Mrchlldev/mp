# Marcellino F.A - Designer Portfolio

Portfolio Designer menggunakan Next.js, TypeScript, Tailwind CSS, dan komponen ala Shadcn UI NeoBrutalism.

## Jalankan Project

```bash
npm install
npm run dev
```

Buka `http://localhost:3000`.

## Build

```bash
npm run build
npm run start
```

## Upload ke GitHub

```bash
git init
git add .
git commit -m "Initial portfolio"
git branch -M main
git remote add origin https://github.com/USERNAME/NAMA_REPO.git
git push -u origin main
```
